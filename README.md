# Personal Developer Portfolio

A modern, highly responsive personal portfolio website built as Task 3 for the CodeAlpha Frontend Development Internship.

## ✨ Key Features
* **Custom UI Design:** Features a unique, premium dark-mode aesthetic highlighted by vibrant purple-to-magenta linear gradients and glowing hover states.
* **Dynamic Animations:** Utilizes Vanilla JavaScript to implement a command-line style typing effect in the hero section and smooth scroll-reveal animations using the Intersection Observer API.
* **Fully Responsive:** Built with modern CSS Flexbox and Grid to ensure perfect structural alignment and content stacking across all desktop, tablet, and mobile devices.
* **Semantic Structure:** Organized with clean, accessible HTML5 tags detailing technical skills, academic background, featured projects, and freelance experience.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom CSS variables, linear gradients, backdrop-filters, and media queries.
* **JavaScript (ES6+):** DOM manipulation, timing events, and scroll observation.
